## Project Overview

Czech declension practice app — a PWA that quizzes users on Czech grammatical cases using a database of ~50k words. Built with React 19, TypeScript, Vite, and Tailwind CSS 4.

## Commands

- `npm run dev` — start Vite dev server
- `npm run build` — TypeScript check + Vite production build (output: `dist/`)
- `npm run lint` — ESLint with zero warnings allowed
- `npm run preview` — preview production build locally

No test framework is configured.

## Architecture

**Two-view app** with no router library. `AppContainer` manages a `view` state (`"welcome"` | `"practice"`) and switches between `WelcomeScreen` and `PracticeScreen`.

**Data flow:** AppContainer owns all shared state (selected cases, scores, database) and passes it down via props/callbacks. No global state library — pure React hooks.

**Persistence:** `LocalData` class wraps IndexedDB (via `idb` library) with LocalStorage fallback. Stores user's selected cases and scores.

**Word database:** `database/words.json` (~2.6MB) is dynamically imported at runtime. Generated by `database/scrape_db.py` (takes ~1 hour to run). Each word entry has 7 grammatical cases × 2 numbers (singular/plural), with multiple accepted forms per slot.

**Case numbering:** Cases are indexed 1–14 (7 cases × 2 numbers). Index 0 is unused. Nominative singular (index 1) is excluded from practice since it's the dictionary form.

## Key Files

- `src/components/AppContainer.tsx` — root component, state management, data loading
- `src/components/PracticeScreen.tsx` — quiz UI, puzzle generation, answer checking
- `src/components/WelcomeScreen.tsx` — case selection, statistics display
- `src/components/LocalData.ts` — IndexedDB/LocalStorage persistence
- `src/components/types.ts` — core TypeScript interfaces (IWordDatabase, ICase, ICurrentPuzzle)
- `src/components/constants.ts` — case names, prepositions
- `src/index.css` — Tailwind config with custom theme colors

## Conventions

- Node 24 (see `.nvmrc`)
- All components are flat in `src/components/` — no nested directories
- Named exports only (no default exports — enforced by ESLint)
- Underscore prefix for intentionally unused variables (`_arg`)
- Prefer native JS over lodash utilities (ESLint restricts `_.extend`, `_.isNull`, `Object.assign`)
- Custom Tailwind theme colors: primary (#2B95D6), success, danger, feminine (#ff2d55), masculine (#007aff), neutrum (#8e8e93)
